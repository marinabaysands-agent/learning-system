# Learning System — Design Document v2

## 设计目标

一个人的学习书房。安静、温暖、有质感。像翻开一本装帧精良的书，不像打开一个 SaaS dashboard。

## 参考

- **Readwise Reader** — 卡片节奏、阅读进度的优雅处理、信息密度恰到好处
- **Literal.club** — 书籍封面卡片的呼吸感、typography 的考究
- **paco.me** — 极简个人站的气质、留白的克制
- **Apple Books** — 卡片圆角、阴影层级、字体选择

## 设计原则

1. **内容优先** — 标题是主角，meta 信息退后
2. **呼吸感** — 宁可少放，不要挤
3. **一致性** — 全站一个卡片语言、一个间距系统、一个色彩逻辑
4. **安静的层级** — 用字号和字重区分层级，不靠颜色喊叫

## 色彩系统

```
背景:     oklch(98.5% 0.003 60)   — 极淡暖灰，像未漂白的纸
卡片:     oklch(100% 0 0)         — 纯白
文字主:   oklch(15% 0.01 60)      — 接近纯黑但带一丝暖
文字次:   oklch(50% 0.01 60)      — 中灰
文字弱:   oklch(65% 0.01 60)      — 浅灰（meta、时间）
强调色:   oklch(45% 0.10 30)      — 暖棕红（仅用于 hover、链接、进度条）
边框:     oklch(92% 0.003 60)     — 极淡分隔线
```

标签色: 统一用文字色 + 浅色背景圆角 pill，不用彩色背景。
- 标签背景: oklch(94% 0.01 hue)
- 标签文字: oklch(40% 0.04 hue)
- 这样标签有颜色区分但不会刺眼

## 字体系统

```
标题:   'Noto Serif SC', Georgia, serif  — 中文衬线，安静有力
正文:   'Source Sans 3', system-ui, sans-serif
```

字号阶梯（1.25 大三度）:
- 页面标题: 2rem / 700
- 卡片标题: 1.125rem / 600
- 正文: 1rem / 400
- Meta/标签: 0.8125rem / 400
- 辅助文字: 0.75rem / 400

行高: 标题 1.3, 正文 1.75

## 间距系统（8px base）

```
4px   — 最小间距（标签之间）
8px   — 紧凑间距
16px  — 元素内间距
24px  — 卡片内 padding
32px  — 卡片之间 gap
48px  — 区块之间
64px  — 页面 section 之间
```

## 卡片设计

```
圆角:     16px
内边距:   24px 28px（桌面） / 20px（移动端）
阴影:     0 1px 3px oklch(0% 0 0 / 0.04), 0 4px 12px oklch(0% 0 0 / 0.06)
Hover:    translateY(-2px) + 阴影加深到 0.10
```

卡片内部结构（Library）:
```
┌──────────────────────────────────┐
│                                  │
│  标题（衬线体，最多2行）          │
│  副标题（1行，浅灰色）           │
│                                  │
│  作者 · 来源 · 时长     [标签]   │
│                                  │
└──────────────────────────────────┘
```

关键：标题在最上面、最大。Meta 信息沉底。

## 页面布局

### 首页
- 标题 + 一句话描述
- 继续阅读（卡片，最多3个）
- 推荐阅读（卡片，最多3个）
- 导航入口（3个文字链接，不需要卡片包裹，简单就好）

### Library（内容库）
- 桌面: 2列网格（max-width 960px）
- 宽屏(1400px+): 3列
- 平板: 2列
- 手机: 单列
- 筛选器: pill 按钮行

### 文章页
- 不变，当前阅读体验已经不错

### 笔记页
- 每篇文章一个白色卡片
- 卡片内：文章标题 + 高亮列表

### 主题页
- 标签云：flex-wrap，每个标签是 pill 样式
- 点击标签 → 展开文章列表（dropdown 或跳转到 library 筛选）

## 响应式断点

```
手机:     < 768px    — 单列，padding 20px
平板:     768-1199px — 2列，max-width 880px
桌面:     1200px+    — 2列或3列，max-width 1100px
```

## 交互

- 卡片 hover: 上浮 + 阴影加深 + 标题变暖棕色
- 链接 hover: 颜色变为 accent
- 筛选器 active: 深色填充
- 进度条: 4px 高，accent 色填充
- 划线工具栏: 已有，保持

## Anti-patterns（不做）

- ❌ 彩色标签背景（用浅色+深文字代替）
- ❌ Emoji 做导航图标
- ❌ 统计数字占首屏
- ❌ 阴影过重（no drop-shadow > 16px blur）
- ❌ 卡片内信息过密
